25.02.2016 Reflections

Can you think of an analogy to describe HTML, CSS and the DOM? How would you describe it to your non-technical friend?

HTML is the barebones, CSS is what makes them look pretty, and the DOM is a combonation of the two as the browser would interpret them..I think.

What is meant by boxifying design?

To break up a design brief into possible HTML box elements, this can be done literally, by drawing boxes.

What is the box model?

A model that describes every HTML element. These are made of padding, margins and the content.
